Sorting ======= There will come a time when you will need to list and sort resources. Hyde allows you to walk the site tree and sort the resources by the predefined settings in your configuration file. You can list and sort by name all your content files. * [About](/about.html) \ (about.html) * [First steps](/first-steps.html) \ (first-steps.html) * [Grouper](/advanced/grouper.html) \ (grouper.html) * [Index](/index.html) \ (index.html) * [Overview](/advanced/overview.html) \ (overview.html) * [Sorter](/advanced/sorter.html) \ (sorter.html) * [Tagger](/advanced/tagger.html) \ (tagger.html) Or list only those in the current node (folder). In this case that would be all advanced topics. 1. [Overview](/advanced/overview.html) 2. [Sorter](/advanced/sorter.html) 3. [Grouper](/advanced/grouper.html) 4.
